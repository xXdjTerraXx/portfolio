# 🌐 Interactive Personal Website

An interactive personal website that acts as the central hub for everything I create — web development, game design, music, and my day-to-day presence on the internet.

---

## ✨ Overview

This project is more than a traditional portfolio — it’s a living, evolving space that reflects what I’m building, listening to, feeling, and working on in real time.

Instead of static pages, the site functions as a **personal platform** where visitors can explore interactive systems, follow updates, and engage with both my work and my current state of mind.

It blends:
- 🎮 Game-like interactivity (PixiJS-powered systems)
- 💬 Real-time communication (live chat)
- 🎵 Music integration (custom player + listening activity)
- 🧠 Personal expression (status updates, mood tracking, blog posts)
- 🌐 Live data integrations (APIs for real-world + personal data)

---

## 🧠 Personal CMS & Live Presence

At the core of the site is a custom-built **content management system (CMS)** that allows me to update different parts of the site in real time.

This powers:

### 💭 Status Updates
- Short-form posts (similar to tweets)
- Used to share thoughts, updates, and what I’m currently doing

### 😊 Mood & Mood Tracker
- Current mood displayed on the site
- Historical mood tracking over time
- Adds a more human, personal layer to the experience

### 📝 Blog Posts
- Longer-form writing about development, music, and ideas
- Devlogs and reflections

### 📸 Photo Album
- A growing collection of images
- Helps document life and creative inspiration

Together, these systems make the site feel alive — not just something I built once, but something I continuously update.

---

## 🌐 Live Data Integrations

The site connects to external APIs to reflect real-time information:

- 🎧 **Last.fm Integration**
  - Displays what I’m currently listening to
  - Shows recent tracks and listening activity

- 🌦️ **Weather API**
  - Displays current weather conditions
  - Adds environmental context to the site

These integrations help bridge the gap between the digital space and real life.

---

## 🚀 Features

### 💬 Real-time Chat System
- Built with Socket.IO
- Live messaging between users
- Server-side message history loading

### 🎮 Interactive Canvas
- Powered by PixiJS
- Animated sprites and dynamic environments
- Cursor-reactive elements (including a cat that follows you 👀)

### 🎵 Custom Audio Player
- Built with Howler.js
- Multiple track selection
- Play, pause, and switching support

### 🧩 Game Development Integration
- Systems and concepts from my game *Barista Hell*
- Reusable animation, movement, and collision logic

### 🔧 Backend API
- Node.js + Express
- Handles chat, CMS data, and external integrations
